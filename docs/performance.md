## Reproducing the Benchmark

1. Start RabbitMQ locally on `127.0.0.1:5672`.
2. Build CosmoBroker in release mode and enable AMQP:

```
COSMOBROKER_ENABLE_AMQP=true \
COSMOBROKER_AMQP_PORT=5679 \
dotnet run --project CosmoBroker.Server/CosmoBroker.Server.csproj --configuration Release
```

3. In a separate shell run:

```
bash benchmarks/compare_amqp.sh
```

The script starts CosmoBroker with a dedicated AMQP port, runs the `RabbitMQ.Client` comparison harness, streams results to `benchmarks/amqp-compare-report.txt`, and prints a concise summary to the terminal.

## Extended Matrix

For wider coverage across publish/consume profiles and payload sizes run:

```
dotnet run --project CosmoBroker.Benchmarks/CosmoBroker.Benchmarks.csproj -- \
  --mode compare-amqp-matrix \
  --profile quick \
  --cosmo-url amqp://guest:guest@127.0.0.1:5679/ \
  --rabbit-url amqp://guest:guest@127.0.0.1:5672/
```

Adjust `--profile` or `--count` to tune runtime and repeatability.
